# SEO PACKAGE

## Synopsis

The SEO Package is a bundle containing all of the SEO-related data collected by the app. The AI uses the SEO package with the selected Writer Model and Brief to generate the story content.

The SEO package includes: 

- All data collected from the user during the New Project setup process:
    - H1/Title/Headline
    - Primary Keyword
    - Secondary Keyword
    - User-Written Story Topic

The SEO Package is also updated in real time with:
- The keywords selected by the user from the Suggested Keywords in the SEO Wizard tool on the right sidebar.
