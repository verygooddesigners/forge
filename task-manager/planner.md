# PLANNER

This is your master document for tracking active tasks and bugs.

Items move here from Inbox after being assigned task IDs. Work progresses through status sections: PENDING to IN PROGRESS to COMPLETED.

---

## PENDING

Instructions:
- Bugs listed first (treated as highest priority)
- Tasks sorted by priority (5 stars to 1 star)
- Same priority items sorted by task ID (oldest first)

---

## IN PROGRESS

Instructions:
- Usually 1-2 items actively being worked on
- Move here when starting work on a task/bug
- Update as work progresses

---

## BLOCKED

Instructions:
- Items waiting on external input or dependencies
- Include note about what is blocking progress
- Move back to PENDING when unblocked

---

## NOTES

- Task IDs are assigned sequentially and never change
- Bugs have no priority level (always treated as highest priority)
- When completed, items move to completed.md with notes and timestamp
- Keep this file clean - only active work should be listed here
